Microsoft Excel > Shortcuts

Shortcuts are key combinations that allow you to complete a task without having to use your mouse to find it in the task bar or otherwise. They are extremely useful staying focused on what you are working on rather than fiddling with the software to acheive something. These are some basic shortcuts that you find yourself using often.

Ctrl + a = Select all. This will select everything on the sheet all at once
Ctrl + b = Bold. This will bold the data of whatever space you have selected.
Ctrl + c = Copy. This will copy whatever is currently selected and hold it on your clipboard. Note that when using shortcuts, your clipboard can only hold one item at a time.
Ctrl + d = Fill Down. This take the data from the cell directly above your currently highlighted cell and copies it to your currently highlighted cell
Ctrl + e = Flash Fill. This takes a sequence of data and copies the whole column in the column next to it. This one is a bit tricky to get to work effectively. I doubt you will get much use out of this one but I listed it in case someone could find an efficient use for it.
Ctrl + f = Find. This will open a small window that will allow you to type a piece of data and search your document for matching data. Press Enter to cycle through all iterations and Esc to exit the window.
Ctrl + g = Goto. This will open up a window that will allow you to enter a coordinate and automatically move you to that coordinate. For example A1 will take you to the very top left of the spreadsheet.
Ctrl + h = Replace. This will open up a window that, like the Find function, will ask you for data to find. You will also be able to input replacement data. This function will allow you to find all pieces of a certain data point and replace it with new data. For example someones name or an error.
Ctrl + i = Italicize. This will italicize the data of whatever space you have selected
Ctrl + n = New. This will open a new and blank document.
Ctrl + s = Save. This will save your document. Spam this when working so you do not lose your progress!
Ctrl + u = Underline. This will underline the data of whatever space you have selected.
Ctrl + v = Paste. This will paste or enter whatever is being held in your clipboard into the currently selected space
Ctrl + x = Cut. This will cut and remove whatever is currently selected and hold it on your clipboard.
Ctrl + z = Undo. This will undo the last edit that you did on your current sheet. You can keep going back multiple times to undo multiple edits. The memory for this function is only 16 edits however you can edit your registry to extend it up to 100.

Here are a few more shortcuts that will help you format your data without the need of browsing through windows.

Ctrl + Shift + @ = Time Format. Input this combination over a cell with data to format it as a time stamp.
Ctrl + Shift + # = Date Format. Input this combination over a cell with data to format it as a date
Ctrl + Shift + $ = Currency Format. Input this combination over a cell with data to format it as a measure of currency. This will only use your native currency on your keyboard or operating system.
Ctrl + Shift + % = Percentage Format. Input this combination over a cell with data to format it as a percentage. Decimal places will be Excels defualts however.
Ctrl + Shift + ^ = Exponential Format. Input this combination over a cell with data to format it as an exponent. For example 123456789 will become 1.23E+08.
Ctrl + Shift + ~ = General Format. Input this combination over a cell to return it to it's original defualt formatting.

Now here are some really incredible shortcuts that will allow you to parse through the taskbar without using your mouse!

Alt + w = Open "View" tab in the top taskbar. From here you will be able to do TONS of things without using your mouse. For example when you want to freeze or unfreeze panes you can do Alt + w > F > F. Once you memorize all of the options opened up from this you will be a spreadsheet genius
